# Лабораторная работа № 2 - Машинное обучение на больших данных (Apache Spark, SparkML)

## Датасет
NYC Yellow Taxi Trip Data (тот же, что в ЛР № 1).

## Задачи
- Регрессия: предсказание суммы поездки `total_amount`.
- Бинарная классификация: были ли чаевые (`high_tip = tip_amount > 0`).

## Модели (по варианту = номер в группе mod 4)
Задаются переменной `VARIANT` в начале каждого ноутбука.

## Структура
- `1__Regression.ipynb` - подготовка признаков, обучение и валидация модели регрессии (RMSE, R²).
- `2__Classification.ipynb` - балансировка классов, обучение и валидация классификатора (Precision, Recall, ROC-AUC), сохранение модели в HDFS.

## Запуск
1. Подключиться к кластеру (см. инструкцию из ЛР № 1).
2. Открыть ноутбуки, выбрать окружение `pyspark-env`.
3. Поставить свой `VARIANT` и выполнить ячейки по порядку.
